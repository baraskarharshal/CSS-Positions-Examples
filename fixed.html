1. Fixed positioned element is removed from normal flow.

2. Width and height is automatically adjusted to fit content.

3. If left,right,top and bottom values are not given then element is placed with respect to its immediate parent.

4. If left,right,top and bottom values are given then element is placed with respect to viewport (viewport is visible area in browser window).

5. Fixed element will not move along with horizontal or vertical scroll unlike relative/absolute position.
