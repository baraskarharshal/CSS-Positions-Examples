1. left, right, top and bottom values does not affect position of Sticky element.

3. Sticky element is always placed with respect to its immediate parent element.

4. If there is scroll then on scroll, element will move along with scroll until it comes at the edge of viewport. Eg. for vertical scroll, element will stick at the top of viewport even if user scrolls all the way to the bottom. Same for horizontal scroll.
